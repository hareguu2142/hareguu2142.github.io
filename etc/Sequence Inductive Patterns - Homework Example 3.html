수열1: 발산하는 등차수열 수열2: 수렴하는 망원급수

귀납적 패턴 분석

수열1: 발산하는 등차수열 \(a_n = n + 2\)

수열의 정의와 항

이 수열의 일반항은 \(a_n = n + 2\)로 정의됩니다. 각 항은 이전 항보다 1만큼 큽니다. 이러한 수열을 공차가 1인 등차수열이라고 부릅니다.

첫 10개 항을 나열하면 다음과 같습니다:

\(a_1=3, a_2=4, a_3=5, a_4=6, a_5=7, a_6=8, a_7=9, a_8=10, a_9=11, a_{10}=12, \dots\)

수열의 변화

\(n\)이 커짐에 따라 항의 값(\(a_n\))도 계속해서 커집니다. 항의 값이 특정 상수에 가까워지지 않고 무한히 증가하므로 이 수열은 끝없이 커집니다.

수학적으로는 \(\lim_{n \to \infty} (n+2) = \infty\) 와 같이 표현합니다.

부분합 분석

부분합 \(S_N\)은 첫째항부터 \(N\)번째 항까지의 합을 의미합니다. 즉, \(S_N = \sum_{n=1}^{N} a_n\) 입니다.

1항부터 10항까지의 합 (\(S_{10}\)):

\[ S_{10} = \sum_{n=1}^{10} (n+2) = \left(\sum_{n=1}^{10} n\right) + \left(\sum_{n=1}^{10} 2\right) \] \[ = \frac{10(10+1)}{2} + (2 \times 10) = 55 + 20 = 75. \]

일반적인 N항까지의 합 (\(S_N\)):

\[ S_N = \sum_{n=1}^{N} (n+2) = \left(\sum_{n=1}^{N} n\right) + \left(\sum_{n=1}^{N} 2\right) \] \[ = \frac{N(N+1)}{2} + 2N = \frac{N^2+N+4N}{2} = \frac{N^2 + 5N}{2}. \]

부분합 \(S_N\)은 \(N\)에 대한 2차식으로, \(N\)이 커질수록 그 값 또한 무한히 커집니다. 따라서 이 수열의 급수(무한합)는 끝없이 커집니다.

수열2: 수렴하는 망원급수 \(a_n = \frac{1}{(n+1)(n+2)}\)

수열의 정의와 항

이 수열의 일반항은 \(a_n = \frac{1}{(n+1)(n+2)}\)로 정의됩니다. 각 항은 인접한 두 항의 곱의 역수로 표현됩니다. 이 수열의 항은 부분분수 분해를 통해 합이 소거되는 망원급수(telescoping series)의 형태를 가집니다.

첫 10개 항을 나열하면 다음과 같습니다:

\(a_1=\frac{1}{6}, a_2=\frac{1}{12}, a_3=\frac{1}{20}, a_4=\frac{1}{30}, a_5=\frac{1}{42}, a_6=\frac{1}{56}, a_7=\frac{1}{72}, a_8=\frac{1}{90}, a_9=\frac{1}{110}, a_{10}=\frac{1}{132}, \dots\)

수열의 변화

항의 값 \(a_n = \frac{1}{(n+1)(n+2)}\)은 \(n\)이 커짐에 따라 분모가 무한히 커지므로, 항의 값은 0에 가까워집니다.

수학적으로는 \(\lim_{n \to \infty} \frac{1}{(n+1)(n+2)} = 0\) 입니다.

부분합 분석 (망원급수)

이 수열의 부분합 \(S_N = \sum_{n=1}^{N} \frac{1}{(n+1)(n+2)}\)은 다음과 같이 부분분수 분해를 통해 쉽게 계산할 수 있습니다.

먼저 일반항을 부분분수로 분해합니다:

\[ a_n = \frac{1}{(n+1)(n+2)} = \frac{1}{n+1} - \frac{1}{n+2} \]

부분합 \(S_N\)은 다음과 같이 소거됩니다:

\[ S_N = \sum_{n=1}^{N} \left( \frac{1}{n+1} - \frac{1}{n+2} \right) \] \[ S_N = \left( \frac{1}{2} - \frac{1}{3} \right) + \left( \frac{1}{3} - \frac{1}{4} \right) + \dots + \left( \frac{1}{N+1} - \frac{1}{N+2} \right) \] \[ S_N = \frac{1}{2} - \frac{1}{N+2} \]

부분합 \(S_N\)은 \(N\)이 커질수록 \(\frac{1}{N+2}\) 항이 0에 수렴하므로, 급수는 특정 값인 1/2에 가까워집니다.

수학적으로는 \(\lim_{N \to \infty} S_N = \lim_{N \to \infty} \left( \frac{1}{2} - \frac{1}{N+2} \right) = \frac{1}{2} - 0 = \frac{1}{2}\) 입니다.

부분합의 수렴 과정

부분합 \(S_N\)은 1/2에 아래에서부터 증가하며 수렴합니다.

  • \(S_1 = \frac{1}{2} - \frac{1}{1+2} = \frac{1}{2} - \frac{1}{3} = \frac{1}{6} \approx 0.1667\)
  • \(S_2 = \frac{1}{2} - \frac{1}{2+2} = \frac{1}{2} - \frac{1}{4} = \frac{1}{4} = 0.25\)
  • \(S_3 = \frac{1}{2} - \frac{1}{3+2} = \frac{1}{2} - \frac{1}{5} = \frac{3}{10} = 0.3\)
  • \(S_4 = \frac{1}{2} - \frac{1}{4+2} = \frac{1}{2} - \frac{1}{6} = \frac{1}{3} \approx 0.3333\)

1항부터 10항까지 합

\[ S_{10} = \frac{1}{2} - \frac{1}{10+2} = \frac{1}{2} - \frac{1}{12} = \frac{6}{12} - \frac{1}{12} = \frac{5}{12} \approx 0.41666... \]

이 값은 실제 수렴값인 1/2 (0.5)에 근접하고 있음을 알 수 있습니다.
